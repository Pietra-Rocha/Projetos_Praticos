🚀 Embedded C/C++ Starter Projects

Repositório com projetos rápidos, práticos e educativos para desenvolvimento embarcado utilizando C/C++, Arduino, IoT, AWS e Bluetooth.
Este repositório reúne exemplos simples e bem estruturados para acelerar o desenvolvimento de aplicações embarcadas, testes de hardware e prototipagem de soluções IoT.
O objetivo é criar uma coleção organizada de projetos base, que podem ser reutilizados como pontos de partida para sistemas maiores.


📚 Conteúdo do Repositório
Aqui você encontrará exemplos como:
🔌 Comunicação Bluetooth
☁️ Integração com AWS IoT
📡 Projetos de Internet das Coisas (IoT)
⚡ Projetos básicos com Arduino
📟 Comunicação entre dispositivos
🔧 Testes rápidos de sensores e atuadores
📊 Coleta e envio de dados

Cada projeto possui:
Código simples e direto
Estrutura organizada
Comentários explicativos
Fácil adaptação para novos projetos

C
C++

Este repositório foi criado para:
📚 Aprendizado prático de sistemas embarcados

🧠 Melhorar habilidades em C/C++ embarcado
